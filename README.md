## プロダクト説明
二度付けの成功には、店主の動きを見極めること非常に重要です。誘惑に負けないよう心を落ち着かせ、背徳感を楽しみながら二度付けに挑戦しましょう。店主は非常に怖いです。くれぐれも店主に見つかり店の出禁にならないよう細心の注意を払いましょう。
一度出禁になったら、もう二度とその串カツを味わえないかも...
